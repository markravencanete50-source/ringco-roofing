Finish setup

The dashboard shell is ready. A few steps connect sign-in and live data:

  1. Connect Firebase. In the Firebase console → Project settings → Your apps → Web app, copy the config object and paste it into FIREBASE_CONFIG in /js/admin.js.
  2. Turn on email/password login. Firebase → Authentication → Sign-in method → enable Email/Password. Then Authentication → Users → Add user with your admin email and a password. Make sure a /users doc exists with that same email, role: "admin", status: "active", and publish the rules from /firestore.rules.
  3. Turn on website tracking. Create a free Google Analytics 4 property, copy its Measurement ID (G-XXXXXXXXXX), and paste it into /js/analytics.js.
  4. Build the dashboard. In Looker Studio, create a report on that GA4 property, then Share → Embed report and paste the URL into LOOKER_EMBED_URL in /js/admin.js.

Keep the Looker report shared only to your Google account(s) so the data stays private. Send me the Firebase config, GA4 ID, and Looker URL and I’ll drop them in for you.
