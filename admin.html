The dashboard shell is ready. Two quick steps connect it to live Google data:

  1. Turn on website tracking. Create a free Google Analytics 4 property for ringcoroofing.com, copy its Measurement ID (G-XXXXXXXXXX), and paste it into /js/analytics.js. Visits start recording immediately.
  2. Build the dashboard. In Looker Studio, create a report on that GA4 property (or start from the “GA4” template). Then Share → Embed report, copy the URL from the embed code, and paste it into LOOKER_EMBED_URL at the top of /js/admin.js.
